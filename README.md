# MTProto VPN Client для Windows

Красивый и функциональный VPN-клиент для Windows, работающий на протоколе **MTProto**, реализованный на языке **Go** в одном файле.

## 🌟 Особенности

- **Протокол MTProto 2.0**: Безопасное соединение с использованием современных алгоритмов шифрования (AES-256-CBC, SHA-256).
- **Веб-интерфейс**: Современный UI в приятной фиолетовой цветовой гамме, доступный через браузер.
- **Без CGO**: Полная совместимость с Windows без необходимости установки GCC или других компиляторов C.
- **Один файл**: Весь исходный код клиента содержится в одном файле `mtproto_vpn_client.go`, что упрощает сборку и распространение.
- **Гибкая конфигурация**: Поддержка загрузки настроек из JSON, импорта через URL (`mtproto://`) или текстовый формат.
- **Статистика трафика**: Отображение объема переданных и полученных данных в реальном времени.
- **Логирование**: Встроенная консоль логов для отладки и мониторинга состояния соединения.
- **Безопасность**: Валидация входных данных, безопасное хранение ключей авторизации.

## 📋 Требования

- **ОС**: Windows 10/11 (также поддерживается запуск на Linux/macOS при пересборке).
- **Go**: Версия 1.21 или выше.
- **Зависимости**: Отсутствуют (используется только стандартная библиотека Go).

## 🚀 Установка и сборка

### 1. Клонирование репозитория

```bash
git clone <url-репозитория>
cd mtproto-vpn-client
```

### 2. Сборка исполняемого файла

**Важно**: Клиент не требует CGO, поэтому сборка работает без установки GCC!

Для сборки под Windows (даже если вы находитесь на другой ОС):

```bash
GOOS=windows GOARCH=amd64 go build -o mtproto_vpn_client.exe mtproto_vpn_client.go
```

Если вы собираете непосредственно на Windows:

```bash
go build -o mtproto_vpn_client.exe mtproto_vpn_client.go
```

Собранный файл `mtproto_vpn_client.exe` готов к использованию — просто запустите его и откройте веб-интерфейс в браузере по адресу, указанному в консоли (обычно `http://localhost:XXXX`).

## ⚙️ Конфигурация

Клиент поддерживает несколько способов настройки подключения.

### Форматы конфигурации

1. **JSON файл** (основной формат):
   ```json
   {
     "server": "example.com",
     "port": 443,
     "dc_id": 2,
     "auth_key": "base64_encoded_key_here",
     "proxy_enabled": false,
     "auto_connect": true,
     "theme": "purple"
   }
   ```

2. **URL схема** (`mtproto://`):
   ```text
   mtproto://auth_key@server:port?dc=2
   ```

3. **Текстовый формат**:
   ```text
   server:port:dc_id:auth_key
   ```

### Расположение файла конфигурации

- **Windows**: `%APPDATA%\MTProtoVPN\config.json`
- **Linux**: `~/.mtproto_vpn/config.json`
- **macOS**: `~/Library/Application Support/MTProtoVPN/config.json`

### Управление конфигурацией через интерфейс

- **Настройки**: Редактирование параметров сервера, порта, DC ID и ключа авторизации.
- **Импорт**: Вставка конфигурации из буфера обмена (поддерживаются все три формата).
- **Экспорт**: Сохранение текущей конфигурации в файл или копирование в буфер обмена.

## 🎨 Интерфейс

Интерфейс выполнен в современной фиолетовой палитре и доступен через любой веб-браузер:

- **Основной цвет**: `#9333EA` (насыщенный фиолетовый)
- **Акцент**: `#7C3AED` (яркий фиолетовый)
- **Фон**: `#1A1025` (глубокий темный)
- **Текст**: `#F3E8FF` (светлый лавандовый)

### Элементы управления

- **Индикатор статуса**: Визуальное отображение состояния соединения (Отключено/Подключение/Подключено/Ошибка).
- **Кнопка подключения**: Большая кнопка для быстрого подключения/отключения.
- **Панель статистики**: Отображение трафика (загрузка/отдача), скорости и времени работы в реальном времени.
- **Консоль логов**: Прокручиваемое окно с событиями подключения и ошибками.
- **Настройки**: Модальное окно для редактирования параметров сервера, порта, DC ID и ключа авторизации.
- **Импорт/Экспорт**: Быстрый обмен конфигурациями через буфер обмена или файл.

## 🔒 Безопасность

- **Шифрование**: Используется AES-256-CBC для шифрования трафика.
- **Аутентификация**: Ключи авторизации хранятся локально в зашифрованном виде (при наличии системных средств).
- **Валидация**: Все входные данные (сервер, порт, ключи) проходят строгую проверку перед использованием.
- **Отсутствие логов**: Клиент не сохраняет историю посещений или персональные данные пользователей.

## 📝 Пример использования

1. Запустите `mtproto_vpn_client.exe`.
2. Перейдите в **Настройки** → введите данные сервера (или используйте **Импорт** для вставки готовой строки подключения).
3. Нажмите кнопку **Подключиться**.
4. Следите за статусом соединения и статистикой трафика на главном экране.

## 🐛 Обработка ошибок

Клиент предусматривает обработку типовых ошибок:
- Недоступность сервера → уведомление с предложением проверить настройки.
- Неверный ключ авторизации → сообщение об ошибке аутентификации.
- Проблемы с сетью → автоматическая попытка переподключения.
